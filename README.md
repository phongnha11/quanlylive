PBC Digital Hub - Hệ thống Phân tích Giáo dục
Ứng dụng được xây dựng cho Trường THPT Phan Bội Châu (Phan Thiết) nhằm mục đích chuyển đổi số trong công tác quản lý chất lượng học sinh.
Tính năng
Real-time Dashboard: Phân tích phổ điểm ngay lập tức.
Cảnh báo sớm: Phát hiện học sinh có nguy cơ rớt hạng hoặc học lệch.
Tương thích Excel: Hỗ trợ nhập liệu từ file Excel chuẩn của nhà trường.
Công nghệ sử dụng
Python 3.10
Streamlit 
CloudPandas & Plotly
Tác giả
Tổ Công tác Chuyển đổi số - THPT Phan Bội Châu.
